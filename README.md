# Student Grade Calculator

A simple Python program that calculates a student's current school grade and university year based on their birth year. It also predicts future education status based on custom logic (6-year university duration).

## Features
- Calculates current age and school grade.
- Determines if the student is in primary/high school, university, or graduated.
- **Future Prediction:** Calculates what grade the student will be in X years later.
- Custom logic for 6-year university duration and can easily be changed through code

## How to Run
1. Just run the code in python
2. Enter your birthday and how many years future you want to calculate
